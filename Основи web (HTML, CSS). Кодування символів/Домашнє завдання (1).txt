3)Перевірка відображення HTML-документа в різних браузерах:


* Chrome:
- Усі стилі відображаються коректно.
- Inline, internal та external стилі працюють без відмінностей.


*  Firefox:
- Без помилок, але трохи інший рендеринг шрифтів (Arial виглядає менш жирно).
- Колір у <div> може виглядати менш контрастним.


*  Microsoft Edge:
- Подібне відображення до Chrome.
- У зовнішньому CSS при швидкому відкритті сторінки може бути коротка затримка завантаження стилю.






*  Safari (macOS):
- Всі стилі працюють, але розміри шрифтів можуть трохи відрізнятись.
- Інколи інтерпретація margin/padding відрізняється.


Висновок: відмінності мінімальні, але шрифти та відступи варто тестувати у всіх браузерах.
